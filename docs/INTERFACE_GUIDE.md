# RAG评估系统 - 系统界面说明

## 评测界面功能详解

### 左侧菜单栏
- **测试集管理** - 创建和选择测试集
- **新建按钮** - 快速创建新的测试集

### 主界面标签页

#### 1. 编辑测试集
- **向量数据加载** - 从JSON文件加载向量数据
- **测试用例表格** - 显示所有测试用例
- **Ground Truth Chunks** - 选择关联的Chunks
- **新增用例按钮** - 添加新的测试用例

#### 2. Excel导入
- **文件上传** - 上传包含测试用例的Excel文件
- **数据预览** - 显示导入的数据和错误信息
- **确认导入** - 批量导入测试用例

#### 3. RAG接口测试
- **配置参数** - Base URL、API Key、Workspace ID
- **测试问题** - 输入要测试的问题
- **测试结果** - 显示RAG系统的回答
- **源文档表格** - 显示检索到的源文档
  - 文档ID - 唯一标识符
  - 标题 - 源文档的标题
  - 相关性分数 - 检索相关性评分
  - 摘要 - 文档内容摘要
  - 完整文本 - 点击展开查看完整内容

#### 4. RAG评测
- **配置参数** - RAG系统连接参数
- **开始评测** - 执行评测任务
- **评测指标** - 显示各项评测指标
  - 正确性 - 模型答案与真实答案的相似度
  - 忠实度 - 答案是否基于源文档
  - 相关性 - 答案是否回答了问题
  - 连贯性 - 答案的逻辑清晰度
- **详细结果** - 每个用例的详细评测结果

#### 5. 数据构建
- **文档输入** - 输入要生成测试数据的文档
- **目标大小** - 设置生成的数据集大小
- **构建结果** - 显示生成的数据统计
  - 总数据量 - 生成的总数据条数
  - 难度分布 - easy/medium/hard的分布
  - 类型分布 - 不同问题类型的分布
  - 来源分布 - Synthetic/Real/Adversarial的分布
- **数据预览** - 显示生成的数据样本

## 关键指标说明

### 评测指标统计
- **总用例数** - 参与评测的总用例数
- **成功用例** - 成功完成评测的用例数
- **平均指标** - 各项指标的平均值
- **综合评分** - 0-100的综合评分

### 详细结果表格
- **问题** - 测试用例的问题
- **正确性** - 答案正确性评分
- **忠实度** - 答案忠实度评分
- **相关性** - 答案相关性评分
- **连贯性** - 答案连贯性评分
- **耗时** - 系统响应时间

## 使用技巧

1. **快速查看完整信息** - 点击表格行展开查看详细内容
2. **批量操作** - 使用Excel导入快速添加多个测试用例
3. **自动生成数据** - 使用数据构建模块自动生成高质量测试数据
4. **追踪源文档** - 在RAG接口测试中查看检索到的源文档
5. **对比分析** - 在评测结果中对比不同用例的表现

## 常见问题

**Q: 如何添加Ground Truth Chunks?**
A: 先加载向量数据，然后在创建用例时从下拉列表中选择Chunks

**Q: 如何查看源文档的完整内容?**
A: 在RAG接口测试的源文档表格中，点击行展开查看完整文本

**Q: 如何批量导入测试用例?**
A: 切换到"Excel导入"标签页，上传包含问题和参考答案的Excel文件

**Q: 如何自动生成测试数据?**
A: 切换到"数据构建"标签页，输入文档内容并设置目标大小，点击"构建数据集"
